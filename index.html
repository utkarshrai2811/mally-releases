Apple menu, then About This Mac. If the Chip line says Apple (for example Apple M2), choose Apple Silicon.

Apple Silicon

Go to the latest release

First launch: if macOS says "Mally is damaged"

It is not damaged. Mally is not signed with a paid Apple certificate yet, so macOS blocks apps it cannot verify after a download. Open it once like this:

  1. In the warning, click Cancel. Do not click Move to Bin.
  2. Open the Terminal app (hold Command, press Space, type Terminal, press Return).
  3. Paste this line and press Return:
$ xattr -dr com.apple.quarantine /Applications/Mally.app

Then open Mally normally. You only do this once per install. The command removes the hidden "downloaded from the internet" tag that triggers the block.
